1. Google Summer of Code with MBDyn (2024 & 2025)

I was selected as a Google Summer of Code contributor for the MBDyn open-source project for two consecutive years. I successfully completed my 2024 project and am currently developing new features for 2025.

Completed Project (GSoC 2024): Building a Modern Python Preprocessor

My primary goal was to build a user-friendly Python library to make creating complex engineering simulations in MBDyn easier. The complete work of 2024 can be found here. My key contributions included:

  • Expanded Core Functionality: I extended the library to support all major MBDyn entities (like nodes, elements, and drivers), making it a comprehensive tool for building a wide variety of models.
  • Improved Model Control: I developed new Python classes to manage simulation settings, giving users precise control over initial conditions and runtime behavior.
  • Created a Model Container: I designed a central "Model" class to hold all parts of a simulation, organizing everything into a single, clean, and easy-to-manage object.
Ongoing Project (GSoC 2025): High-Level Tools & Blender Integration

My current work focuses on making the preprocessor even more powerful and accessible:

  • Rotorcraft Modeling API: I am transforming a command-line tool for modeling rotorcraft (like helicopters and drones) into a simple Python API. This will allow users to build these complex models with intuitive commands instead of verbose configuration files.
  • Blender Integration: I am integrating the Python preprocessor directly into Blendyn (a Blender addon). This will create a seamless, all-in-one environment where users can build, run, and visualize advanced simulations without ever leaving Blender.

Technologies used: Python, Object-Oriented Programming, Blender, YAML, MBDyn

2. Book Recommendation System
